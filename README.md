## How to

Run main

1. Create virtualenv
2. Set your env variable for your mysql db credentials (username, password) in activate.bat
e.g.
set DB_USER=root
set DB_PASS=pass
3. activate virtualenv
4. pip install -r requirements.txt
5. python -m app.main

Run tests

python -m pytest -s
